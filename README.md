# Seminar Registration App - UTS Pemrograman Mobile 1

Aplikasi Pendaftaran Seminar sederhana yang dibangun menggunakan **Kotlin**, **Android Studio**, **View Binding**, dan **Material Design 3**. Proyek ini dibuat untuk memenuhi tugas UTS Pemrograman Mobile 1.

## 🚀 Fitur Utama
- **Login & Register**: Sistem autentikasi sederhana dengan validasi input.
- **Dashboard**: Halaman utama yang menyapa pengguna dan akses ke formulir.
- **Form Pendaftaran**: Input data peserta (Nama, Email, No HP, Gender) dan pilihan seminar.
- **Validasi Real-time**: Mengecek validitas email, format nomor HP, dan kelengkapan data sebelum submit.
- **Konfirmasi Dialog**: Menggunakan `MaterialAlertDialogBuilder` untuk verifikasi data.
- **Halaman Hasil**: Menampilkan ringkasan data yang telah diinput pengguna.

## 🛠️ Teknologi yang Digunakan
- **Bahasa**: [Kotlin](https://kotlinlang.org/) (2.0.21)
- **UI Framework**: [XML Layouts](https://developer.android.com/guide/topics/ui/declaring-layout) & [Material Components](https://material.io/components)
- **Binding**: [View Binding](https://developer.android.com/topic/libraries/view-binding)
- **Minimum SDK**: API 24 (Android 7.0)
- **Target SDK**: API 35 (Android 15)
- **Build System**: Gradle Kotlin DSL

## 📸 Struktur Folder Utama
- `app/src/main/java/.../`: Berisi logika aktivitas (`LoginActivity`, `MainActivity`, `FormActivity`, dll).
- `app/src/main/res/layout/`: Berisi desain antarmuka pengguna (XML).
- `app/src/main/res/values/`: Berisi pengaturan warna, dimensi, tema, dan string.

## ⚙️ Cara Menjalankan Proyek
1. Clone repository ini ke lokal.
2. Buka proyek menggunakan **Android Studio Ladybug** atau versi yang lebih baru.
3. Tunggu proses **Gradle Sync** selesai.
4. Hubungkan emulator atau perangkat fisik Android.
5. Klik tombol **Run (Shift + F10)**.
